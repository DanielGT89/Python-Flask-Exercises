Este repositório contém uma coleção de projetos desenvolvidos durante meu processo de aprendizagem em desenvolvimento web utilizando Python, Flask, Bootstrap e PostgreSQL. O objetivo deste material é consolidar conhecimentos fundamentais envolvendo rotas, templates, formulários, integração backend/frontend e banco de dados. Cada aplicação foi construída com foco em prática incremental, evoluindo desde páginas HTML simples até sistemas interativos com armazenamento de informações, validação de dados e interface responsiva.

Os projetos presentes aqui representam exercícios aplicados utilizando conceitos essenciais como requisições HTTP (GET e POST), lógica de programação, estilização com Bootstrap, manipulação de dados e princípios de desenvolvimento web moderno. Este repositório também serve como base de referência e evolução técnica para estudos futuros, facilitando melhorias contínuas, versionamento e boas práticas com Git e GitHub.
